# 01 - Enlaces y Navegación 
Este fue uno de mis primeros proyectos donde aprendí a estructurar la navegación de un sitio web utilizando etiquetas `<a>` y rutas relativas.

## Lo que aprendí:
* Conexión entre diferentes archivos HTML.
* Atributos de enlaces (target, href).
* Estructura básica de un sitio multi-página.
